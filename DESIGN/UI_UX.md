- Main yard view + HUD

- Inspector panel (selected machine)

- Rotate (R)
- Confirm (LMB), cancel (RMB/ESC)

- Input buffer count
- Output buffer count
- For Splitter: filter selection (checkbox list of item categories)

## Overlays

- Blocked outputs highlight
- Storage fullness heatmap (optional)
