# Milestones

## M0 — Repo Bootstrap
- Engine chosen, project boots, renders isometric grid, basic input
- Data-driven definitions load (items/machines/recipes)

## M1 — Vertical Slice (10–15 minutes)
- Delivery system + yard inventory
- Conveyors, splitter, magnet, washer, shredder, smelter, storage, seller
- 4 scrap categories + 3 products
- Contracts: accept/complete/fail, money rewards, backlog penalty
- Basic UI: placement, inspect machine, throughput overlay, contract panel

## M2 — Depth Pass
- Contamination levels + jams
- Supplier profiles + daily forecast
- 5–8 upgrades + research tree

## M3 — Content Expansion
- More scrap types + electronics/hazard
- More machines (optical sorter, eddy current)
- More products + quality thresholds
